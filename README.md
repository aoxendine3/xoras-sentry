# Integrity Sentry Core
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Institutional-grade environment variable auditing and secret detection.**

`integrity-sentry` is a high-fidelity security utility designed for the top 1% of the global development ecosystem. It utilizes a **Hybrid AST Engine** to provide **Deterministic Structural Integrity** where traditional regex tools fail.

## Key Features
- 🚀 **Hybrid AST Analysis**: Comprehends code structure (Acorn-powered).
- 🔒 **Zero-Trust Security**: Constant-time cryptographic verification.
- ⚡ **O(1) Memory Stability**: Line-by-line streaming with hard-limit guards.
- 🛠️ **Institutional Governance**: Audit-ready telemetry and tamper-evident locking.

## Quick Start
```bash
npx integrity-sentry .
```
