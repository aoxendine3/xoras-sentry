# Security Policy

## Supported Versions

We are committed to securing the `integrity-sentry-core` ecosystem. The following versions are currently receiving security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do NOT open a public issue. Instead, follow these steps:

1. **Email us**: Send a detailed report to security@integrity-sentry.com (Placeholder) or via the GitHub Security Advisory portal.
2. **Details**: Include a proof of concept, the impact, and any remediation suggestions.
3. **Timeline**: We aim to acknowledge receipt within 24 hours and provide a resolution within 7 days.

## Our Commitment to Integrity

This project utilizes a **Hybrid AST Engine** and **Constant-Time Cryptography** to prevent timing attacks and data leakage. All reported vulnerabilities are handled with immediate priority.
